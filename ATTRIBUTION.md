This project uses open geospatial data. The code license (MIT) does **not**
apply to the data; each data source keeps its own license.

## OpenStreetMap

- **Source**: OpenStreetMap, retrieved via the Overpass API
- **License**: [Open Data Commons Open Database License (ODbL) 1.0](https://opendatacommons.org/licenses/odbl/)
- **Credit**: © OpenStreetMap contributors — https://www.openstreetmap.org/copyright
- **Important**: The processed files under `data/processed/` and
  `web/public/data/` (e.g. `city.json`) are **derivative databases** of
  OpenStreetMap data. Under ODbL §4.4, these derivative databases are also
  made available under **ODbL 1.0**. If you reuse `city.json` or data derived
  from it, you must attribute OpenStreetMap and share derived databases under
  ODbL as well.
- Retrieval date and the exact Overpass query are recorded in
  `data/processed/meta.json` (generated by the pipeline).

## In-app credit

The web demo displays "© OpenStreetMap contributors" (and PLATEAU / GSI
credits once those datasets are used) in a credits overlay reachable from the
on-screen "©" button.

## Policy on real-world storefronts / 実在店舗の扱い

Building shapes, heights and placement are reproduced from open data.
Trademarks, logos and copyrighted signage of real shops are **not**
reproduced faithfully; the visual phase (Phase 4) will only evoke the
atmosphere (colors / abstract shapes), not actual brand assets.
