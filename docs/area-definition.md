# 対象エリア定義（正式版）

本プロジェクトの対象エリア・座標系・ローカル原点の正式定義。
パイプライン（`scripts/`）とランタイム（`web/`）はすべてこの定義に従う。

## 中心点（ローカル原点）

| 項目 | 値 |
|---|---|
| 地点 | 自由が丘駅（東急東横線・大井町線） |
| WGS84 | **35.6075 N, 139.6690 E** |
| EPSG:6677（平面直角座標系 第IX系, JGD2011） | easting (Y) = **-14888.490 m**, northing (X) = **-43533.196 m** |

- 系原点: 緯度36°N, 経度139°50′E（139.833333…°）, 縮尺係数 0.9999, GRS80
- proj4定義: `+proj=tmerc +lat_0=36 +lon_0=139.833333333333 +k=0.9999 +x_0=0 +y_0=0 +ellps=GRS80 +units=m +no_defs`

## ローカル座標系（ランタイム用）

- 原点 = 自由が丘駅（上記）
- `local_x = easting − (−14888.490)`（東 = +X）
- `local_y = northing − (−43533.196)`（北 = +Y）
- Three.js（Y-up・右手系）へのマッピング: `three.x = local_x`, `three.z = −local_y`, `three.y = 標高（MVPでは0）`
- 単位: メートル
